# AstrBot 天气查询插件

AstrBot 插件，用于查询全球城市的当前天气信息及未来天气预报。

## 功能特点

- 支持查询全球任意城市的**当前天气**和**未来天气预报**
- 无需 API Key，使用 wttr.in 免费天气服务
- 支持命令方式：`/weather 北京`、`/天气 上海`、`/forecast 北京 3`
- 支持自然语言（LLM 智能识别）：`北京天气怎么样`、`查询上海未来3天天气`
- 异步 HTTP 请求，不阻塞机器人响应
- 显示温度、湿度、风速、能见度、气压等详细信息
- 天气预报包含最高/最低温度、天气状况、降雨概率、日出日落时间

## 使用方法

### 方法 1：命令方式（传统）

**查询当前天气：**
```
/weather <城市名>
/天气 <城市名>
/查天气 <城市名>
/tq <城市名>
```

**查询未来天气：**
```
/weather <城市名> <天数>
/天气 <城市名> <天数/明天/后天>
/forecast <城市名> [天数]
/天气预报 <城市名> [天数]
```

### 方法 2：自然语言（智能识别）

直接发送包含天气相关词汇的消息，插件会自动识别：

```
北京天气怎么样
查询上海的天气
明天温度多少
北京未来3天天气预报
```

### 示例

**当前天气：**
```
/weather Beijing
/天气 上海
/查天气 Tokyo
/tq London
```

**未来天气：**
```
/weather 北京 3          # 查询北京未来3天天气
/天气 上海 明天          # 查询上海明天天气
/forecast 广州 2         # 查询广州未来2天天气
/天气预报 深圳           # 查询深圳未来3天天气（默认）
```

**自然语言方式：**
```
北京天气怎么样
查询上海的天气
今天天气如何
明天北京天气
未来3天广州天气如何
```

## 返回信息

### 当前天气
- 地点名称
- 天气状况描述
- 当前温度和体感温度
- 湿度百分比
- 风速和风向
- 能见度
- 气压
- 观测时间

### 天气预报
- 日期和星期
- 天气状况描述
- 最高/最低/平均温度
- 降雨概率
- 湿度和风速
- 日出日落时间

## 安装方法

1. 将插件文件夹复制到 AstrBot 的 `data/plugins/` 目录下
2. 重启 AstrBot 或在 WebUI 中重载插件
3. 插件会自动安装所需的依赖

## 依赖项

- `aiohttp>=3.8.0`

## 技术说明

- 使用 wttr.in API 获取天气数据（支持3天预报）
- 所有网络请求均为异步，使用 aiohttp
- 内置超时和错误处理机制
- 支持中英文城市名查询
- 天气预报自动选取正午时段作为当日天气代表
