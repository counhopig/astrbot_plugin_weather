# AstrBot 天气查询插件

> 🤖 **本插件的全部代码、架构设计、功能规划与文档，均由 AI（GitHub Copilot · Claude Sonnet）全程管理和撰写，作者仅提供产品方向与验收。**

AstrBot 插件，用于查询全球城市的当前天气信息及未来天气预报。

支持两种天气数据提供商：
- **wttr.in**：免费，无需 API Key
- **彩云天气**：更精准，支持分钟级降水预报，需要 API Key

## 功能特性

- 支持查询全球任意城市的**当前天气**和**未来天气预报**
- 支持命令方式：`/weather 北京`、`/天气 上海`、`/forecast 北京 3`
- 支持自然语言（LLM 智能识别）：`北京天气怎么样`、`查询上海未来3天天气`
- 异步 HTTP 请求，不阻塞机器人响应
- 显示温度、湿度、风速、能见度、气压等详细信息
- 天气预报包含最高/最低温度、天气状况、降雨概率、日出日落时间
- 彩云天气额外提供：空气质量（AQI/PM2.5）、生活指数

## 安装

1. 将插件文件夹复制到 AstrBot 的 `data/plugins/` 目录下
2. 重启 AstrBot 或在 WebUI 中重载插件
3. 插件会自动安装所需的依赖

## 配置项

在 AstrBot 管理后台的插件配置页面中进行配置：

| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| `weather_provider` | string | `"wttr"` | 天气数据提供商：`wttr` 或 `caiyun` |
| `caiyun_api_key` | string | `""` | 彩云天气 API Key / Token |
| `caiyun_api_version` | string | `"v2.6"` | 彩云天气 API 版本 |

### 通过命令配置

```
/setweather wttr                    # 切换到 wttr.in（免费）
/setweather caiyun <your_api_key>   # 切换到彩云天气
```

### 获取彩云天气 API Key

1. 访问 [彩云天气开放平台](https://dashboard.caiyunapp.com/)
2. 注册并登录
3. 在「API 凭证管理」中获取 Token
4. 免费版每天有 1000 次调用限制

## 命令

| 命令 | 别名 | 说明 |
|------|------|------|
| `/weather <城市名>` | `/天气` `/查天气` `/tq` | 查询当前天气 |
| `/weather <城市名> <天数>` | `/天气 <城市名> <天数/明天/后天>` | 查询未来天气 |
| `/forecast <城市名> [天数]` | `/天气预报` | 查询未来天气预报 |
| `/setweather <提供商> [key]` | - | 设置天气数据提供商 |

### 使用示例

**当前天气：**
```
/weather Beijing
/天气 上海
/查天气 Tokyo
/tq London
```

**未来天气：**
```
/weather 北京 3          # 查询北京未来3天天气
/天气 上海 明天          # 查询上海明天天气
/forecast 广州 2         # 查询广州未来2天天气
/天气预报 深圳           # 查询深圳未来3天天气（默认）
```

### 自然语言方式

直接发送包含天气相关词汇的消息，插件会自动识别：
```
北京天气怎么样
查询上海的天气
今天天气如何
明天北京天气
未来3天广州天气如何
```

## 返回信息

### 当前天气
- 地点名称
- 天气状况描述
- 当前温度和体感温度
- 湿度百分比
- 风速和风向
- 能见度
- 气压
- 观测时间

### 彩云天气额外信息（当使用彩云天气时）
- 空气质量（AQI、PM2.5）
- 生活指数（紫外线、舒适度）

### 天气预报
- 日期和星期
- 天气状况描述
- 最高/最低/平均温度
- 降雨概率
- 湿度和风速
- 日出日落时间

## 依赖

- `aiohttp>=3.8.0`

## 技术说明

- **wttr.in**：免费天气服务，支持 3 天预报
- **彩云天气**：高精度天气预报，地理位置精确到一公里，时间分辨率精确到一分钟
- 所有网络请求均为异步，使用 aiohttp
- 内置超时和错误处理机制
- 支持中英文城市名查询
- 使用 Nominatim (OpenStreetMap) 进行城市名到经纬度的转换（彩云天气需要）
